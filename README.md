# Sungjae Kim's Blog

개인 블로그 프로젝트입니다.

## 기술 스택

### Frontend
- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **React Router** - 라우팅
- **SCSS** - 스타일링

### 배포
- **GitHub Pages** - 정적 사이트 호스팅
